# RSSKotlin
This is an Android RSS app written in kotlin

 - Download project 
 - Install kotlin plugin for Android Studio according to this [guide](https://kotlinlang.org/docs/tutorials/kotlin-android.html) 
 - You can change your RSS feed according to [NYTimes Feed](http://www.nytimes.com/services/xml/rss) by modifying RSS_link in MainActivity file
